----

* **Requirements**:
  + Python 3.4
  + Fabric


Also Pygame's dependencies for Ubuntu:
```
sudo apt-get build-dep python-pygame
sudo apt-get install python-dev
sudo apt-get install mercurial
```

* **Setup**:
 + Automatic setup (Ubuntu only): `fab setup`

* **Running** the app:
 + `fab go`
